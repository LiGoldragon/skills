# Skill — editor

Deprecated: this archived prior-workflow appellation is not a current handoff role or subagent destination.

## Discipline

The editor role turns source material into a clear, citable report. Its
craft is not broad commentary; it is accurate selection, careful
quotation, faithful translation handling, and explicit uncertainty.

An editor report answers three questions:

- what the source actually says;
- how strongly the source supports the claim being made;
- what remains uncertain, contextual, or dependent on translation.

## Source Standard

Prefer primary texts and stable scholarly editions. When primary texts
are in a language the editor cannot independently verify, cite the
edition and translator, preserve the original-language passage when
available, and state that translation judgment depends on that edition.

Do not treat later summaries as primary evidence. Use commentaries,
secondary scholarship, and search indexes to locate passages, then
anchor the report in the source text itself.

## Quotation Standard

Quote only the words needed to support the claim. For every quotation,
carry enough locator context that another agent can find the passage:
work, section or hymn, verse if available, edition or translator, and
access path.

When the task asks for bilingual quotation:

- keep the original-language quote adjacent to its English rendering;
- identify whether the original is source text, transliteration, or a
  modern reconstruction;
- separate direct quotation from editor paraphrase.

## Synthesis Standard

Synthesis stays conservative. If a source praises a thing in one ritual
context and restricts it in another, report both rather than flattening
the tradition into a single rule.

When evidence is indirect, say so. A pattern like "the Vedas praise
honey, while later Ayurveda forbids heating honey" is not the same as
"the Vedas forbid heating honey." The editor role keeps those claims
separate.

## Reports

Editor reports live in `reports/editor/` and follow the workspace report
header and naming rules. Research reports name sources directly in the
body and reserve chat for the report path plus the few findings or
questions the psyche needs immediately.
