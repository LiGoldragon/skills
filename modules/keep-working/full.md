# Skill — keep working

## Rules

Mid-task psyche messages are usually added context, not a stop signal. Read them, integrate them, and keep working.

Only an explicit stop, wait, cancel, or redirect changes the execution path. A redirect changes the work; it does not make the agent idle.

Capture or route any durable intent the message carries when that is part of the assigned work. Correct course if the new information changes the task.

Ask only when intent is genuinely unclear or contradictory. Do not ask whether to continue merely because a message arrived.
